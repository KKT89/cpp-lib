## リファレンス生成方法

[MkDocs](https://www.mkdocs.org/)を用いて作成しています。

### セットアップ

```shell
$ uv venv .venv
$ source .venv/bin/activate
$ uv pip install -r requirements.txt
```

以下のコマンドが正しく実行出来ればOKです。

```shell
$ mkdocs serve
```
